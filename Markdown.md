# Syntax

- ## <ins>**'#'** symbol: </ins> 

    - ### This symbol is used to create a heading <ins>**the more '#' you add the smaller it gets**</ins>
        - #### Name + Number of '#' symbol
        - # Title1
            - ## Title2    
                - ### Title3
                    - #### Title4
                        - ##### Title5
                            - ###### Title6   
    > Note: cannot do the following: `####### Title7`. Since the maximum number is 6.
